Potenzreihe der Funktion $f(x)$ um den Punkt $x_o$:
\mathbox{
    f(x) = \sum_{n=0}^{\infty} a_n \cdot (x-x_o)^n
}
\begin{itemize}
    \item Höchstens eine Potenzreihe von $f$ um $x_o$ existiert.
    \item Konvergiert für $\abs*{x-x_o} < r$
\end{itemize}
